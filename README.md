# Delicious Bites Restaurant Website

A modern, responsive restaurant website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful and clean interface with smooth animations
- **Interactive Menu**: Filter menu items by category (Appetizers, Main Courses, Desserts, Drinks)
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Contact Form**: Functional contact form (ready for backend integration)
- **Mobile-Friendly Navigation**: Hamburger menu for mobile devices

## Sections

1. **Hero Section**: Eye-catching landing area with call-to-action
2. **About Section**: Information about the restaurant
3. **Menu Section**: Interactive menu with filtering options
4. **Contact Section**: Contact information and contact form
5. **Footer**: Additional links and social media

## How to Use

1. Simply open `index.html` in a web browser
2. No build process or dependencies required
3. All files (HTML, CSS, JavaScript) are self-contained

## Customization

- Edit `index.html` to change content
- Modify `style.css` to change colors, fonts, and styling
- Update `script.js` for additional interactivity

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
